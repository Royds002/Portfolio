Uno

Uno is een in-vehicle infotainment system die ervoor zorgt dat 50-plussers zonder afleiding van het IVI veilig op hun werk aankomen met de auto.



Opdrachtgever

WeAreReasonablePeople

Jaar

2021 - Leerjaar 2 CMD

De opdracht

De opdrachtgever voor dit project was WeAreReasonablePeople. WeAreReasonablePeople kwam naar ons toe met de opdracht om een infotainment system te ontwerpen waarbij de knoppen overbodig zijn en de bestuurder alle functies op één plek kan bereiken, zonder dat er wordt ingeleverd op gebruiksvriendelijkheid. Ik moest bij deze opdracht het communicatie systeem designen van het infotainment systeem

Iteratie 1

Om een zo goed mogelijk ontwerp te maken ben ik begonnen met het maken van een flowchart om de flow van het systeem te bepalen. Van daaruit ging ik wireframes maken om te kijken hoe ik het systeem ging vormgeven. Het eerste prototype dat ik ging maken was een paper prototype. Hierbij ging ik verschillende schermen maken en deze heb ik later ook getest bij de doelgroep. Voordat ik bij de doelgroep ging testen heb ik eerst een testplan opgesteld over hoe ik het testen ging aanpakken.



Iteratie 2

Bij de tweede iteratie heb ik de feedback die ik had gekregen tijdens het testen verwerkt in nieuwe wireframes. Dit deed ik omdat uit de tests was gebleken dat er nog een paar schermen veranderd moesten worden. Toen de feedback was verwerkt in wireframes heb ik mijn wireframes uitgewerkt in Figma en heb ik er een low fidality clickable prototype van gemaakt. Na het maken van mijn prototype heb ik het weer getest bij doelgroep. Hierdoor kreeg ik weer feedback dat ik weer kon verwerken in mijn volgende prototype.



Iteratie 3

Na de tests van iteratie 2 heb ik een paar iconen aangepast omdat deze niet heel duidelijk waren. Ook heb ik kleur toegevoegd aan mijn prototype deze heb ik hetzelfde gemaakt als bij mijn teamgenoten. Verder had ik het prototype nog verder uitgebreid met meerdere schermen. Ik had deze iteratie ook weer het prototype getest met de doelgroep om te kijken of ze het nog steeds goed vonden.



Iteratie 4

Voordat ik dit project kon afronden had ik nog wat feedback verwerkt in mijn prototype. Zo heb ik nog verschillende features toegevoegd aan het prototype. Ook heb ik mijn prototype samengevoegd met die van mijn andere teamgenoten zodat het een geheel werd. Om dit project af te ronden heb ik samen met mijn team een presentatie gehouden voor de opdrachtgever.
